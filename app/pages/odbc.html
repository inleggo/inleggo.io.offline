ODBC

Conecte su equipo a un orígen de datos ODBC.

Crear conexiones a orígenes de datos ODBC

Para poder conectarse desde Inleggo.io DB (MySQL) a un equipo con Windows necesitamos un conector ODBC.

Descarga directa

Descargar el driver de conexión:

Driver 32 bits
Driver 64 bits

Instale el driver MySQL









Cree la conexión ODBC

Dirijase al panel de control:





Seleccione la versión (32 o 64) que instalo.


Seleccione la opción agregar.


Seleccione la opción MySQL ODBC 5.3 Unicode Driver.


Parámetros de configuración para MRE 2016

Data Source Name: mre_2016_temp
TCP/IP Server: inleggo.io
Port: 3306
User: mre
Password: 20131380101
Database: inleggo_mre_2016_temp



Probamos la configuración con el boton test y luego le damos en ok.


Como resultado final veremos el nombre del orígen de datos creado.
